# bigodeStore

A **bigodeStore** é um projeto de front-end que simula o design de uma loja virtual (E-commerce) voltada para a venda de jogos digitais, atuando conceitualmente como uma parceira oficial do ecossistema Xbox. O projeto foi estruturado utilizando práticas de semântica HTML5, organização de seções de conteúdo e integrações visuais modernas.

## 🎯 Objetivos do Projeto

Através do desenvolvimento desta página, foram aplicados e consolidados os seguintes conceitos de desenvolvimento web:
1. **Estrutura Semântica Avançada:** Divisão de layouts complexos usando contêineres e identificadores apropriados (`id` e `class`) para organizar o fluxo de leitura da página.
2. **Integração de Recursos Externos:** Utilização de fontes tipográficas externas (Google Fonts para as famílias *Inter* e *Lexend*) e vetores SVG para ícones de interface.
3. **Seções Dinâmicas de E-commerce:** Arquitetura de componentes típicos de lojas virtuais, incluindo carrosséis promocionais com sobreposição de camadas (*overlays*), grades de gêneros de jogos, seções de *Best Sellers* e listas de títulos populares com destaque para preços originais, valores promocionais e descontos percentuais.
4. **Área de Lançamentos e Pré-vendas (*Coming Soon*):** Organização de layouts assimétricos com tamanhos de cartões diferenciados para priorização visual de produtos.
5. **Prova Social e Avaliações:** Criação de componentes dedicados a revisões e depoimentos de usuários (*User Reviews*), simulando métricas de satisfação através de sistemas de estrelas e avatares.
6. **Rodapé Corporativo Multilinks:** Construção de um rodapé estruturado dividindo menus institucionais, recursos de suporte, links de políticas e vinculação de redes sociais através de ícones.

---

## 📋 Arquitetura e Componentes da Interface

A página está organizada nas seguintes seções principais:
*   **Header / Navbar:** Menu superior contendo o logotipo da marca, links de navegação (`Games`, `Browse Games`, `Store`) e botões de autenticação para o usuário (`Sign up`, `Login`).
*   **Promotion Carousel:** Banner de destaque principal que exibe títulos em evidência com chamadas para ação (*Call-To-Action*).
*   **Why Us Section:** Bloco de diferenciais competitivos da loja com ícones customizados indicando credibilidade e variedade.
*   **Games Genre Section:** Exibição em grade dos principais gêneros oferecidos (Strategy, FPS, RPG, Puzzle, Sports, etc.).
*   **Product Showcases (Best Seller & Popular):** Exibição de cartões de produtos detalhados que mostram as capas dos jogos, nomes, valores convertidos na moeda fictícia (Rp) e badges promocionais (ex: *Free Download* ou marcadores de desconto).
*   **Coming Soon Section:** Estrutura em blocos para realçar títulos muito aguardados que se encontram em fase de pré-venda.
*   **User Review Section:** Feed com depoimentos de clientes satisfeitos para reforçar a confiabilidade da plataforma.
*   **Footer Section:** Área institucional completa dividida em colunas de recursos, políticas e atalhos para mídias sociais.

## 🛠️ Tecnologias Utilizadas

*   **HTML5** - Estruturação semântica e esqueleto da aplicação web.
*   **CSS3** - (Mapeado via folha de estilos externa) Utilizado para o controle de fontes, paleta de cores escuras e distribuição de componentes.
